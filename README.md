# Stack-and-Qu# Stack and Queue Application (Java)

هذا المشروع هو تطبيق عملي لهياكل البيانات (Data Structures) الأساسية: **الكدسة (Stack)** و **الطابور (Queue)** باستخدام لغة البرمجة **Java**.

## 🚀 المميزات
- تنفيذ كامل لـ **Stack** (LIFO - Last In First Out).
- تنفيذ كامل لـ **Queue** (FIFO - First In First Out).
- مشروع منظم باستخدام **Maven**.
- كود سهل القراءة والفهم للأغراض التعليمية.

## 🛠️ التقنيات المستخدمة
- **Language:** Java
- **Build Tool:** Maven
- **Concepts:** Data Structures (Stack & Queue)

## 📂 هيكلة المشروع
- `Stack.java`: يحتوي على العمليات الأساسية مثل (push, pop, peek).
- `Queue.java`: يحتوي على العمليات الأساسية مثل (enqueue, dequeue).
- `Mavenproject1.java`: الملف الرئيسي لتشغيل وتجربة الكود.

## 💻 كيفية التشغيل
1. قم بتحميل المشروع (Clone).
2. افتحه باستخدام أي IDE (مثل NetBeans, IntelliJ, أو VS Code).
3. قم بعمل Build للمشروع باستخدام Maven.
